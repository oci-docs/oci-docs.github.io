    OCI Infra 기초

    OCI 기초 서비스인 Compute(VM), VCN, Load Balancer 등 IaaS 기본 서비스를 구성하는 핸드온 랩입니다. 이 절에서 다루는 Hands-on Lab은 Oracle Cloud Trial Account로 실습할 수 있도록 구성되었습니다.

    OCI Infra Hands-on Lab

    이번 Hands-on Lab은 Oracle Cloud Free Tier 계정으로 진행할 수 있도록 구성했습니다. 아직 Oracle Cloud 계정이 없는 상태라면, 다음 문서를 참조하여 Oracle Cloud 계정을 만드시기 바랍니다.

    • OCI Free Tier 등록

    추가로 다음 문서의 Hands-on Lab을 완료했고 가이드에 따라 OCI User, Group, Compartment 및 Policy 작업을 마쳤다는 가정으로 진행합니다. 기본 환경 구성을 진행하지 못했다면 다음 Hands-on Lab을 완료해 주시기 바랍니다.

    • OCI 사용자&보안 설정

    Oracle Account 정보

    Oracle Cloud 계정을 다음과 같은 정보로 만들었다는 가정으로 진행하겠습니다.

    항목 설정값
    Account Name ocidemo001
    OCI Admin User taewanme@gmail.com
    Password Welcome123456!
    OCI Account의 Home Region Seoul

    다음 OCI User를 사용합니다.

    OCI User Password 작업 Compartment
    demo.user Welcome123456! root/sandbox/demo-prod

    Home Region Login URL

    위 Oracle Account가 Hands-on Lab에서 사용하는 Region Login URL(로그인 URL)은 다음과 같습니다.

    • https://console.ap-seoul-1.oraclecloud.com/tenate=ocidemo001

    위 Region 로그인 URL은 다음과 같은 패턴으로 만들어집니다.

    https://console.{HOME_RESION_ID}.oraclecloud.com/tenate={Account Name}
    • 사용할 Oracle Account가 Home Region으로 Seoul 리전이 아닌 다른 리전을 사용하실 경우에는 {HOME_RESION_ID}를 사용하시는 Home Region ID로 교체하시기 바랍니다.
      • Region ID 확인는 다음 문서 참조: OCI 리전 정보
    • {Account Name}은 사용하시는 OCI Account ID로 교체하시기 바랍니다.
