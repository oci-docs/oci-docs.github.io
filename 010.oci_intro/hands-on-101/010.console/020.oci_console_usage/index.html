    OCI 햄버그 메뉴

    OCI Console에서 햄버그 메뉴 아이콘을 클릭하여 다음과 같은 메뉴에 접근할 수 있습니다.

    • 코어 인프라
    • Database
    • 데이터 및 AI
    • 솔루션 및 플랫폼
    • 추가 Oracle Clodu 서비스
    • 거버넌스 및 관리

    햄버그 메뉴에 출력되는 모든 메뉴는 OCI Console에 로그인한 사용자가 접근 가능한 서비스 목록입니다. Oracle IDCS와 OCI IAM이 접근 권한을 확인합니다.

    리전 관리

    OCI는 현재(2020년 03월 28일 기준) 16개 글로벌 리전(Gov 리전 제외)을 운영하고 있습니다. OCI Console의 상단 메뉴에서 현재 어떤 리전을 사용하고 있는지 확인할 수 있습니다.

    아래 이미지에서 확인할 수 있는 것처럼, OCI Console의 상단 메뉴에 현재 계정이 사용하고 있는 리전 명이 출력됩니다. 현재 사용하고 있는 계정은 South Korea Central(Seoul) 리전을 사용하고 있는 것으로 표시된 상태입니다.

    South Korea Central(Seoul)을 클릭하며 현재 사용하는 클라우드 계정(Tenante, ocidemo001)이 사용하는 현재 리전과 홈 리전이 출력됩니다. 영역관리(Manage Regions) 메뉴를 클릭하면 리전 관리 페이지로 이동할 수 있습니다.

    영역관리(Manage Regions) 메뉴를 클릭하면 리전 관리 페이지로 이동합니다. 리전 관리 페이지에서는 OCI의 전체 리전 목록이 출력되고, 현재 사용하는 클라우드 계정(ocidemo001)이 사용할 수 있는 리전(구독, Subcribed)을 확인할 수 있습니다. 새로운 리전을 사용해야 한다면, 이 페이지에서 구독 버튼을 클릭해야 합니다.

    지금 Hands-on에서 사용하는 계정은 Oracle Free Trial 계정입니다. Oracle Free Trial 계정은 홈 리전만을 사용하도록 자원 사용이 제한돼 있습니다. 따라서 신규 리전 구독을 시도하면 다음과 같은 에러 메시지가 출력됩니다.

    OCI Cloud Shell

    Cloud Shell은 OCI Console에서 브라우저로 접근하는 VM 인스턴스 터니널입니다. OCI Cloud Shell에는 오라클 클라우드를 사용하는데 필요한 다양한 툴이 설치 및 사전 설정돼 있습니다. Oracle Cloud 사용자는 별도로 클라우드 관리 환경을 구성하고 툴을 설정할 필요 없이 OCI Cloud Shell로 바로 클라우드 관리 작업을 시작할 수 있습니다.

    아래 이미지와 같이 OCI Console에서 OCI CLI를 바로 사용할 수 있습니다. OCI CLI 오라클 클라우드 명령어 기반 관리 툴입니다. OCI Cloud Shell에는 OCI CLI의 완전한 환경 설정이 적용돼 있습니다.

    OCI Cloud Shell에 대한 자세한 내용은 다음 문서에서 확인할 수 있습니다.

    • OCI Cloud Shell: 브라우저 기반 가상 터미널

    고급 검색

    OCI Console은 OCI 인프라스트럭처를 사용하는 모든 자원을 조회하는 SQL과 유사한 형태의 인터페이스를 제공합니다.

    다음 이미지는 OCI Console에서 생성한 모든 자원을 조회하는 쿼리를 실행하는 흐름입니다. 다음 이미지와 같이 OCI Console 상단의 검색 아이콘을 클릭하고 쿼리를 입력하면 관련 자원을 조회할 수 있습니다.

    query all resources

    OCI Console은 고급 검색 예제 쿼리와 함께 고급 검색 쿼리를 실행하는 편집기를 제공합니다. 다음 이미지와 같이 고급 검색을 실행한 상태에서 ‘고급 검색’ 버튼을 클릭하면 고급 검색 편집/실행 페이지로 이동합니다. 이 페이지에서는 다양한 예제 쿼리를 확인할 수 있습니다. OCI Console이 제공하는 예제를 응용하여 다양한 고급 검색 쿼리를 수행하여 OCI 자원 목록을 조회할 수 있습니다.

    고급 검색 쿼리에 대한 문법 및 기능에 대해서는 다음 문서에서 최신 정보를 확인할 수 있습니다.

    OCI Console의 사용자 아이콘에서 사용자 프로파일 정보와 사용자 프로파일 세부 관리 페이지로 이동할 수 있습니다.

    OCI 알림

    OCI Console은 해당 계정(ocidemo001)에 적용되는 시스템 관리 차원의 알람 기능을 제공합니다. 알람 아이콘을 클릭하면 세부 공지 내용을 확인할 수 있습니다. 핸즈-온에서 Oracle Free Trial 계정을 사용하거나 새로 만든 계정에서는 다음과 같이 어떤 내용도 출력되지 않을 수 있습니다.

    실제로 운영 환경으로 사용하는 OCI 계정에서는 알림 메시지가 도착하면 아이콘에 초록색 점이 찍힙니다. 다음과 같은 알림 메시지를 확인할 수 있습니다. 메시지를 유형/기간으로 상세 검색할 수 있습니다.
